Part 3: Object Boundary Detection

Finding exact object boundaries using classical computer vision techniques (no ML/DL).

Methods Implemented:

  • Contour Detection: Otsu thresholding + morphological operations + contour finding
  • GrabCut Segmentation: Iterative graph-cut based foreground/background separation
  • Boundary Shapes: Bounding box, convex hull, minimum enclosing circle, fitted ellipse
