Part 1: Template Matching

Normalized Cross-Correlation
Object Detection & Regional Blurring

Detect objects in scene images using template matching through normalized cross-correlation. Apply Gaussian blur to detected regions using convolution.

Mathematical Foundation:
NCC(x,y) = Σ[T(x',y') × I(x+x',y+y')] / √[Σ T² × Σ I²]
Where T is template, I is image region
  • Multiple template detection
  • Confidence scoring
  • Regional Gaussian blurring
  • Adjustable blur parameters

Part 2: Blur Recovery

FFT & Wiener Deconvolution
Gaussian Blur & FFT Recovery

Apply Gaussian blur through convolution and recover the original image using FFT-based deconvolution with Wiener filtering.

Wiener Filter:
H*(ω) / (|H(ω)|² + λ)
Where H* is conjugate, λ is noise parameter
  • Gaussian convolution
  • FFT-based deconvolution
  • PSNR quality metrics
  • Adjustable blur intensity

Getting Started

Template Matching
  1. Place template images in images/templates/ directory
  2. Upload a scene image containing objects to detect
  3. Configure blur settings (optional)
  4. Run detection to find templates in the scene
  5. View results with bounding boxes and confidence scores
Blur Recovery
  1. Upload any image for blur/recovery demonstration
  2. Adjust Gaussian blur sigma parameter
  3. Run the blur and recovery process
  4. Compare original, blurred, and recovered images
  5. Review PSNR quality metrics
