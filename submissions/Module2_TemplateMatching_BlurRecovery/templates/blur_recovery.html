Algorithm Details

1. Gaussian Blur

Apply 2D Gaussian convolution to create blurred image:

I_blurred = I ⊗ G(σ)
  • Separable convolution for efficiency
  • Kernel size based on 6-sigma rule
  • Maintains image dimensions
2. FFT Processing

Transform to frequency domain for deconvolution:

F{I}, F{G} → Frequency
  • 2D Fast Fourier Transform
  • Kernel padding and shifting
  • Complex number operations
3. Wiener Filter

Optimal recovery with noise suppression:

H*(ω) / (|H|² + λ)
  • Conjugate filter application
  • Noise regularization (λ = 0.01)
  • Inverse FFT for final image
PSNR (Peak Signal-to-Noise Ratio)

Measures recovery quality: PSNR = 20 × log₁₀(MAX / √MSE)
Higher values indicate better recovery quality. Typical good values: 20-40 dB
